## 개요

### TravelZoa 소개
여행의 후기와 정보를 공유하는 웹 커뮤니티 서비스
### 선정이유
여행을 다녀온 후 자신만의 추억을 남기고, 후기나 정보들을 공유해 여행을 준비하는 사람들에게 더 나은 여행을 제공하기 위함
### 주요기능
* 질문 게시판
* 회원가입 / 로그인
* 이미지를 첨부한 후기 / 갤러리 게시판
* 메일 보내기
